# Projeto Contatos

## Sobre o Projeto

Uma aplicação mobile para gerenciamento de contatos, permitindo adicionar e excluir contatos de forma simples e intuitiva. A aplicação integra dados de uma API externa com dados locais armazenados no Firebase Realtime Database.

## Objetivo

Criar uma aplicação que permita aos usuários:

- Visualizar uma lista de contatos com uma API na página listar-contatos
- Listar contatos existente e adicionar novos na página adicionar-contatos
- Script de Build

## Tecnologias Utilizadas

### Frontend

- Ionic Framework
- Angular
- TypeScript
- SCSS

### Backend e Infraestrutura

- Firebase Realtime Database
- API externa (JSONPlaceholder)

### Ferramentas de Desenvolvimento

- Angular CLI
- Ionic CLI
- Visual Studio Code
- Git para controle de versão
